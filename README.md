# -android--navigation-screens-app

## Descrição:
Aplicativo Android desenvolvido em Kotlin utilizando Jetpack Compose para construção de interfaces modernas e declarativas. O projeto demonstra a implementação de navegação entre múltiplas telas usando o componente de navegação do Android Jetpack Navigation.
